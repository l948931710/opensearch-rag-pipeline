| **BM25 (词频)** | 精确关键词兜底 | 91.5% | 单独最弱，但加入三路后 R@1 从 95.7% → 100%（补上 Dense 和 Sparse 的盲区） |

### 8.3 增量价值链

```
Pure Dense (R@1=97.9%, MRR=0.9702)
  │
  ├─ +Sparse → Dense+Sparse (R@1=95.7%, Pollution 43.8%)
  │            R@1 不升反降，但 Pollution 降 2.2%，Margin 升
  │
  ├─ +BM25   → Dense+BM25 (R@1=95.7%, MRR=0.9807)
  │            R@1 不升反降，但 MRR 升 0.0105
  │
  └─ +Both   → D+S+B (R@1=100.0%, MRR=1.0000) ✅
               只有三路同时存在才突破天花板
```

> [!IMPORTANT]
> **核心结论：三路融合的价值不是简单的 1+1+1 叠加，而是三个信号互补覆盖彼此的盲区。**
> - 某些 Query（如精确关键词匹配）只有 BM25 能找到
> - 某些 Query（如语义改写）只有 Dense 能找到
> - Sparse 信号在提升排名精度和降低 Top-5 噪声方面发挥作用
> - 任何单信号或双信号组合都至少丢 1-4 条 Query，只有三路融合达到 R@1=100%
